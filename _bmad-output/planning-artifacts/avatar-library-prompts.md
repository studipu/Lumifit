# Lumifit 36개 아바타 라이브러리 생성 프롬프트

> 작성일: 2026-03-19
> 목적: Nano Banana 기반 36개 대표 체형 아바타 이미지 생성
> 사용 방식: 각 프롬프트를 그대로 복사해 사용

## 정리 기준

- 키와 몸무게는 반드시 숫자와 해석 표현을 함께 넣습니다.
- 프롬프트만으로 체형을 완전히 제어할 수 없으므로, 가능하면 동일한 기준 참조 이미지 1~3장을 함께 넣습니다.
- 36개 아바타는 모두 같은 포즈, 같은 배경, 같은 카메라 거리, 같은 조명으로 생성해야 비교가 가능합니다.
- 얼굴 디테일보다 체형 비율이 잘 드러나는 것이 더 중요합니다.

## 공통 생성 규칙

- 대상: 여성복 중심 MVP용 전신 아바타
- 스타일: 패션 이커머스 비교용 아바타
- 포즈: 정면 전신, 중립 자세
- 배경: 흰색 무지 배경
- 구도: 중앙 정렬, 동일 카메라 거리
- 제약: 액세서리, 가방, 소품, 텍스트, 워터마크 없음
- 중요: 얼굴보다 체형 비율이 잘 보이도록 생성
